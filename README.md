# CRUD Simples com Planilha

Este projeto é um CRUD básico.

## Funcionalidades

- Visualizar dados da planilha
- Adicionar nova linha
- Atualizar linha existente
- Deletar linha

## Requisitos

- Python 3
- pandas
- flask
- openpyxl

## Instale com:

```bash
pip install -r requirements.txt
```

## Criando Planilha ( se não existir )

```bash
python planilha_vazia.py
```

## Executando

```bash
python app.py
```

Acesse a aplicação em `http://localhost:3000`.
